Create cache "configuration" in headlands. Add marina-sample and headlands containers to the same user defined network. Then "headlands" becomes ping-able.

curl -i -XPUT -H"Content-type: application/json" -d'{"storeByValue":true}' http://192.168.99.100:8080/headlands/resources/caches/configuration

Create entry "msg:duke" in the cache "configuration"

curl -i -XPUT -d'duke' http://192.168.99.100:8080/headlands/resources/caches/configuration/entries/msg
